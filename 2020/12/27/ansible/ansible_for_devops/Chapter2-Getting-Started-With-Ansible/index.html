  • Vagrant的几个命令
  1. vagrant halt will shut down the VM
  2. vagrant up will bring it back up,
  3. vagrant destroy will completely delete the machine from VirtualBox.
  4. vagrant up again will re-create it from the base box you originally downloaded
  5. To connect, enter vagrant ssh from the folder where the Vagrantfile is located.
  6. If you want to connect manually, or connect from another application, enter vagrant ssh-config to get the required SSH details.
